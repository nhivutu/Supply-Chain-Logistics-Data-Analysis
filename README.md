# Supply Chain Logistics Data Analysis

Tools: SQL | Excel

## Project Overview

This project analyzes supply chain shipment data to identify logistics trends, supplier performance, and shipping patterns.

## Key Analysis

• Shipment trends over time  
• Top suppliers by shipment volume  
• Shipping mode distribution  
• Average delivery time analysis  

## Key Insights

The analysis highlights which suppliers handle the largest shipment volumes and which shipping methods are most frequently used. The results help support logistics planning and supply chain decision-making.
